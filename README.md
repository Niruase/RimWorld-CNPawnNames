# CN Pawn Names | 简体中文姓名替换模组
适配 RimWorld 1.6 · 基于 Harmony 的轻量化中文姓名优化模组

## 简介
在简体中文环境下游玩环世界时，殖民者、访客、商队、派系角色以及 Backer 付费命名角色仍会显示原生英文姓名，界面观感割裂杂乱。

本模组会在启动时自动检测游戏语言，**仅在简体中文环境下激活**，将原生生成的英文西式姓名统一替换为标准化中文音译姓名，同时清空冗余昵称，保持角色卡片整洁。已有的中文、自定义派系姓名不会被修改，兼顾沉浸感与兼容性。

## 核心特性
- **语言自动判定**：非简体中文环境自动静默停用，无后台性能开销

## 兼容性
- **前置依赖**：Harmony 模组（必须优先加载）
- **适配版本**：RimWorld 1.6 正式版
- **加载顺序**：本模组必须置于 **Harmony 之后**
- 与绝大多数派系扩展、剧情拓展、外观美化模组兼容
- 不覆盖部落、帝国等原版特殊派系的自定义命名规则

## 安装方法
1. 订阅并启用 Harmony 前置模组
2. 将本模组文件夹放入 `RimWorld/Mods` 目录
3. 在模组列表中启用，并调整加载顺序，确保位于 Harmony 下方
4. 将游戏语言切换为简体中文，进入游戏即可生效

## 常见问题排查
### 角色仍然显示英文姓名
- 确认游戏全局语言已设置为简体中文
- 检查 Harmony 是否正常加载、模组有无加载报错
- 打开 `Player.log` 搜索 `[CNPawnNames]` 查看模组启用状态日志

### 模组完全未生效
- 检查加载顺序是否颠倒，必须在 Harmony 之后
- 排查是否存在同类姓名修改模组冲突，可临时禁用其他模组测试

### 游戏出现报错
- 优先排查重度修改角色生成逻辑的模组冲突
- 确认游戏版本与模组适配版本一致
